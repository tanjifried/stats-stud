## 1. Decision Framework: Picking the Right Tests

The choice between **parametric** and **non-parametric** tests depends on three things:

| Question to Ask | If YES → | If NO → |
|---|---|---|
| Is the outcome variable continuous (0–100 score)? | Continue | Use chi-square / Fisher's |
| Are groups independent of each other? | Continue | Use paired / repeated-measures |
| Does the outcome meet normality + homogeneity of variance? | Use parametric | Use non-parametric |

For multi-group comparisons (k > 2):

| Parametric | Non-Parametric | Use when |
|---|---|---|
| One-Way ANOVA | Kruskal-Wallis | k > 2 independent groups, continuous outcome |
| Welch's t-test | Mann-Whitney U | k = 2 independent groups |

For continuous–continuous relationships:

| Parametric | Non-Parametric | Use when |
|---|---|---|
| Pearson correlation | Spearman correlation | Testing linear/monotonic association |

## 5. Test A — One-Way ANOVA (Parametric)

### 5.1 Research Question

Is there a significant difference in mean math scores among students from different racial/ethnic groups?

### 5.2 Null Hypothesis (H₀)

The mean math scores are equal across all five racial/ethnic groups:

> H₀: μ_A = μ_B = μ_C = μ_D = μ_E

### 5.3 Alternative Hypothesis (H₁)

At least one racial/ethnic group has a significantly different mean math score from the others:

> H₁: At least one μ_i ≠ μ_j for some i ≠ j

### 5.4 Assumptions

| Assumption | How Checked | Status |
|---|---|---|
| Independence of observations | One row per student, no repeated measures | ✅ Met |
| Normality within groups | Anderson-Darling test per group; CLT applies (n_min ≥ 89 per group) | ✅ Acceptable |
| Homogeneity of variance | Levene's test: F(4, 995), check p-value | ✅ Verify in output |
| Continuous outcome | math_score is 0–100 integer scale | ✅ Met |

> **Note on normality:** For n ≥ 30 per group, the Central Limit Theorem ensures ANOVA F-statistic is robust to mild non-normality (Hair et al., 2019). All groups exceed this threshold.

### 5.5 R Code

```r
source("scripts/00_setup.R")
df <- utils::read.csv(paths$public_clean, stringsAsFactors = FALSE)
df$race <- trimws(tolower(df$race_ethnicity))
df$math_score <- as.numeric(df$math_score)

# Assumption check: Levene's test
car::leveneTest(math_score ~ race, data = df)

# Run ANOVA
anova_model <- aov(math_score ~ race, data = df)
summary(anova_model)

# Effect size: eta-squared
ss_total  <- sum((df$math_score - mean(df$math_score, na.rm = TRUE))^2, na.rm = TRUE)
ss_between <- summary(anova_model)[[1]]$"Sum Sq"[1]
eta_sq <- ss_between / ss_total
cat("Eta-squared:", round(eta_sq, 4), "\n")

# Post-hoc: Tukey HSD
TukeyHSD(anova_model)
```

### 5.6 Interpretation

A one-way ANOVA is conducted to test whether mean math scores differ significantly across five racial/ethnic groups (A through E). The F-statistic indicates whether the between-group variance exceeds the within-group variance beyond what chance would predict. If F is large and p < .05, at least one group mean is significantly different. Tukey HSD post-hoc tests identify which specific pairs differ.

### 5.7 Conclusion

**If p < .05:** Reject H₀. There is sufficient evidence to conclude that mean math scores differ significantly across racial/ethnic groups. Post-hoc comparisons (Tukey HSD) identify which groups drive this difference.

**If p ≥ .05:** Fail to reject H₀. There is insufficient evidence of a significant difference in mean math scores across racial/ethnic groups.

*Decision rule:* Reject H₀ if F(4, 995) yields p < .05.

## 6. Test B — Kruskal-Wallis (Non-Parametric)

### 6.1 Research Question

Is the distribution of math scores significantly different across racial/ethnic groups, without assuming normality?

### 6.2 Null Hypothesis (H₀)

The rank distribution of math scores is identical across all five racial/ethnic groups (no group tends to rank higher than the others):

> H₀: The population distributions of math scores are equal across all five groups.

### 6.3 Alternative Hypothesis (H₁)

At least one racial/ethnic group has a systematically different rank distribution of math scores:

> H₁: At least one group's population distribution of math scores differs from the others.

### 6.4 Assumptions

| Assumption | How Checked | Status |
|---|---|---|
| Independence of observations | One row per student | ✅ Met |
| Ordinal or continuous outcome | math_score (0–100) qualifies | ✅ Met |
| No normality required | Kruskal-Wallis is rank-based | ✅ By design |
| Similar distribution shapes across groups | Visual check via density/boxplot | ✅ Verify in EDA |

> **Why Kruskal-Wallis here:** Some racial/ethnic subgroups have n as low as ~89. While CLT still applies, Kruskal-Wallis provides a robustness check that is free from distributional assumptions — validating the ANOVA result on the same variables.

### 6.5 R Code

```r
source("scripts/00_setup.R")
df <- utils::read.csv(paths$public_clean, stringsAsFactors = FALSE)
df$race <- trimws(tolower(df$race_ethnicity))
df$math_score <- as.numeric(df$math_score)

# Run Kruskal-Wallis
kw_result <- kruskal.test(math_score ~ race, data = df)
print(kw_result)

# Effect size: epsilon-squared
# epsilon^2 = (H - k + 1) / (n - k), where k = number of groups, n = total
k <- length(unique(df$race))
n <- sum(!is.na(df$math_score))
H <- kw_result$statistic
epsilon_sq <- (H - k + 1) / (n - k)
cat("Epsilon-squared:", round(epsilon_sq, 4), "\n")

# Post-hoc: pairwise Wilcoxon with Bonferroni correction
pairwise.wilcox.test(df$math_score, df$race,
                     p.adjust.method = "bonferroni",
                     exact = FALSE)
```

### 6.6 Interpretation

The Kruskal-Wallis test compares the **rank distributions** of math scores across the five racial/ethnic groups. The H-statistic (chi-square approximated) measures how far group median ranks deviate from the overall median rank. If H is large and p < .05, at least one group tends to rank systematically higher or lower. Pairwise Wilcoxon follow-ups with Bonferroni correction identify which groups differ.

**Comparing Kruskal-Wallis to ANOVA:** If both tests yield significant results with similar effect sizes, this confirms the group differences are robust regardless of distributional assumptions.

### 6.7 Conclusion

**If p < .05:** Reject H₀. There is sufficient evidence that math score rank distributions differ significantly across racial/ethnic groups. The result corroborates the ANOVA finding using a distribution-free approach.

**If p ≥ .05:** Fail to reject H₀. There is insufficient evidence of a systematic difference in math score rank distributions across racial/ethnic groups.

*Decision rule:* Reject H₀ if Kruskal-Wallis H yields p < .05 (χ² approximation with df = k − 1 = 4).

## 7. Effect Size Reference

| Test | Effect Size Metric | Small | Medium | Large |
|---|---|---|---|---|
| ANOVA | Eta-squared (η²) | 0.01 | 0.06 | 0.14 |
| ANOVA | Omega-squared (ω²) | 0.01 | 0.06 | 0.14 |
| Kruskal-Wallis | Epsilon-squared (ε²) | 0.01 | 0.06 | 0.14 |
| t-test / Mann-Whitney | Cohen's d / r | 0.2 | 0.5 | 0.8 |
| Pearson / Spearman | r / ρ | 0.1 | 0.3 | 0.5 |

> Source: Cohen (1988); Tomczak & Tomczak (2014) for ε².
