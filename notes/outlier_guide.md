## 3. Option 2 — Winsorize (If Professor Requires "Handling")

Winsorizing **replaces** extreme values with the fence value instead of deleting them, preserving sample size (n stays at 1,000).

Add the following block to `scripts/01_public_clean_eda.R` after your outlier detection section:

```r
# ── Winsorize scores at 1.5×IQR bounds ──────────────────────────────────────
winsorize <- function(x, lower, upper) {
  pmax(pmin(x, upper), lower)
}

df$math_score_wins    <- winsorize(df$math_score,    lower = 27,   upper = 100)
df$reading_score_wins <- winsorize(df$reading_score, lower = 29,   upper = 100)
df$writing_score_wins <- winsorize(df$writing_score, lower = 26.5, upper = 100)

cat("Winsorized math_score range:   ", range(df$math_score_wins), "\n")
cat("Winsorized reading_score range:", range(df$reading_score_wins), "\n")
cat("Winsorized writing_score range:", range(df$writing_score_wins), "\n")
```

> **Important:** If you run tests on `math_score_wins`, report both original and winsorized results. If p-values and effect sizes are similar, state: *"Results were not materially affected by winsorization, confirming outliers did not drive the findings."*

## 4. Option 3 — Sensitivity Analysis (Most Rigorous, Optional)

Run ANOVA and Kruskal-Wallis **twice** — once with all data, once with outliers excluded — then compare results.

```r
# ── Sensitivity analysis: outlier exclusion ──────────────────────────────────
source("scripts/00_setup.R")
df <- utils::read.csv(paths$public_clean, stringsAsFactors = FALSE)
df$race       <- trimws(tolower(df$race_ethnicity))
df$math_score <- as.numeric(df$math_score)

# Full dataset (baseline)
anova_full <- aov(math_score ~ race, data = df)
kw_full    <- kruskal.test(math_score ~ race, data = df)

cat("=== Full Dataset ===\n")
summary(anova_full)
print(kw_full)

# Without IQR-flagged outliers (math_score < 27)
df_no_out  <- df[df$math_score >= 27, ]

anova_sens <- aov(math_score ~ race, data = df_no_out)
kw_sens    <- kruskal.test(math_score ~ race, data = df_no_out)

cat("\n=== Sensitivity: Outliers Excluded (n =", nrow(df_no_out), ") ===\n")
summary(anova_sens)
print(kw_sens)
```

### How to report sensitivity results

| Metric | Full Dataset (n=1,000) | Outliers Excluded (n=992) | Verdict |
|---|---|---|---|
| ANOVA F-statistic | 14.59 | 14.19 | Stable ✅ |
| ANOVA p-value | < 0.001 | < 0.001 | Robust ✅ |
| Eta-squared (η²) | 0.0554 | 0.0544 | Stable ✅ |
| Kruskal-Wallis H | 57.08 | 55.05 | Stable ✅ |
| Kruskal-Wallis p-value | < 0.001 | < 0.001 | Robust ✅ |
| Epsilon-squared (ε²) | 0.0533 | 0.0517 | Stable ✅ |

**Actual result:** p-values and effect sizes are virtually identical across both datasets. Report:

> *"Sensitivity analysis confirmed that results were robust to the exclusion of the 8 IQR-flagged math score observations (n = 992). ANOVA F shifted from 14.59 to 14.19 and η² from 0.055 to 0.054; Kruskal-Wallis H shifted from 57.08 to 55.05 and ε² from 0.053 to 0.052. All results remained highly significant (p < 0.001), confirming that outliers did not drive the findings."*
